# Project Context

## Produto

VENATTO ERP SaaS e um sistema ERP em modelo SaaS para organizar operacoes da VENATTO.

## Motivo da reconstruucao

A versao anterior ficou complexa demais, perdeu consistencia arquitetural e avancou fases sem validacoes suficientes. Esta reconstruucao prioriza controle de escopo, validacao por fase e simplicidade inicial.

## Direcao do MVP

O MVP sera um projeto unico em Next.js, com backend inicial via Server Actions e API Routes do proprio Next.js. O banco oficial sera PostgreSQL, acessado por Prisma.

## Estado atual

O projeto esta na Fase 0: fundacao documental e tecnica, sem funcionalidades de negocio.
